# Development

## Prerequisites

- Python 3.9+
- [uv](https://github.com/astral-sh/uv) (recommended package manager)

## Development Commands

```bash
# Setup development environment
make setup

# Run tests
make test

# Run tests with coverage
make test-cov

# Code quality checks
make check

# Build documentation
make build

# Serve documentation locally
make serve
```
